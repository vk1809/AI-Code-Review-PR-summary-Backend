AI code Review & PR summary Backend.

--- Think this projects like a Github Copilot PR(Pull-request) Reviewer/ CodeRabiit-lite but Backend only.

Tech Stack

* FastAPI
* Pydantic
* PostgresSQL
* Redis
* LLM (openAI/ Groq)
* Github webHooks
* Background workess
* Docker

Built an AI-powered PR review backend integrating GitHub webhooks and LLMs to summarize code changes, flag risks, and provide automated review comments using async background processing.
